# Movie-Ticket-Booking-System
Java-based movie ticket booking system

## Project Overview
The Movie Ticket Booking System is a **Java-based desktop application** that allows users to browse movies, select show timings, choose seats, and book tickets.  
This project demonstrates Java programming, GUI development using Swing, and database management using MySQL.

---

## Features
- View a list of available movies and show timings
- Select seats from available options
- Book tickets and view booking confirmation
- Simulated payment process
- User-friendly interface using Java Swing

---

## Technologies Used
- **Java** – Core programming language  
- **Java Swing** – For GUI development  
- **MySQL** – Database to store movie and booking details  
- **JDBC** – For connecting Java application with MySQL
